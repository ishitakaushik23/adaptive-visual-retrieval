# Version 1 Goal

Build a semantic visual retrieval system that:

- Accepts an input image.
- Extracts its embedding.
- Compares it with embeddings of all images in the dataset.
- Returns the Top-5 most visually similar images.

## Success Criteria

- User uploads an image.
- System returns meaningful visually similar images.
- Works locally.
- Can later be deployed on Hugging Face.
